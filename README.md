# Tutu - Home Tutoring Marketplace

**Tagline:** "Hey Tutu! Time to Learn."

## Overview
A mobile-first marketplace platform connecting tutors with parents and students seeking personalized home tutoring services.

## Brand Colors
- **Primary:** Indigo (#3A38FF)
- **Accent:** Electric Teal (#00D8C8)

## Design System
- Pill-shaped buttons with full rounded corners
- Super rounded cards (24px+ radius)
- Modern & Trustworthy aesthetic

## Current Status
This repository contains the preserved state of the Tutu app with 11 core screens before full restoration.

## Device Support
- Last 5 generations of iPhone (12-16 series)
- Last 5 generations of Android devices
- Huawei devices with HarmonyOS
- Tablets (iPad, Android tablets)

## Tech Stack
- React + TypeScript
- Tailwind CSS v4.0
- Material Symbols icons
- Mobile-first responsive design
